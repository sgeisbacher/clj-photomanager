Available Functions:
  • /config - shows the current configuration
  • /config/reload - reloads the configuration
  • /cache - shows all cached items
  • /cache/evict/{task} - removes a key {task} from the cache
  • /cache/evict/all - removes all keys from the cache
  • /initialData - fetches initial Data (projects, status, issuetypes)
  • /fetch - fetches file frequency for the given jql query
    Parameters:
    • jql - String, example: "project in (a)"
  • /fetchp - fetches file frequency for the given parameters
    Parameters:
    • projects - String, example: "a, b, c"
    • issuetypes - String, example: "a, b, c"
    • statuses - String, example: "a, b, c"
    • datefrom - String, example: "2015-05-25"
    • dateto - String, example: "2015-05-25"
